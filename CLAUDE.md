## How to help

- Never make an edit that was not explicitly requested.  Never refactor code unless explicitly asked to do so.
- If you detect any issues in existing code, flag them to the user for review.
- Do not pollute this repository with new modules and code.  Specifically: Do not add functions or methods to the core API unless (1) a thorough review reveals that this or similar functionality does not exist yet and (2) the user explicitly authorizes such an edit.
